### Pokedex - Prueba Liverpool

Este es un proyecto de Next.js v13, para inicial el proyecto seguir los siguientes pasos:

```
npm run install
npm run dev
```

Abrir el navegador en [http://localhost:3000](http://localhost:3000) para ver el resultado.
